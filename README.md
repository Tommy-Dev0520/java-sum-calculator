# Java Sum Calculator

## Description

Este proyecto es una calculadora sencilla desarrollada en Java Swing. Permite ingresar dos números, sumarlos y mostrar el resultado en la interfaz gráfica.

## Technologies I used

- Java
- Java Swing
- NetBeans IDE

## How does it work?

- Ingresar dos números.
- Calcular la suma.
- Mostrar el resultado en pantalla.

## What I learned?

Este proyecto fue desarrollado como parte de mi proceso de aprendizaje en Java para practicar:

- Variables
- Conversión de datos (`Integer.parseInt()`)
- Eventos de botones
- Componentes gráficos (JFrame, JTextField, JButton y JLabel)
